# Stashly

A personal image storage solution that allows you to store your images in a secure and easy to use platform.

## Features

- Upload images
- Delete images
- View images
- Star images
- Trash images
- Empty trash
- Download images

## Tech Stack

- Next.js
- Tailwind CSS
- Shadcn UI
- Clerk
- Drizzle ORM
- ImageKit
